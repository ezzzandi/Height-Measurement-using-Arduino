# Height-Measurement-using-Arduino
Measuring height using Arduino UNO

| No. | Components                | Quantity |
|-----|---------------------------|----------|
| 1   | Arduino UNO               |     1    |
| 2   | Breadboard                |     1    |
| 3   | Ultrasonic sensor HC-SR04 |     1    |
| 4   | LCD 16x2 (I2C)            |     1    |
| 5   | Jumper Cable              |     10   |
